使用 tmux

tmux 是一个极其强大的工具，它允许你在服务器后台保持会话活跃，即使 SSH 断开，进程依然在跑。

2. 创建并运行任务

在 VS Code 的终端中：

  1. 新建会话：给任务起个名字（例如 training），方便后续查找。
  2. 运行代码：进入 tmux 界面后，像平常一样运行你的脚本。
  3. 分离会话（Detach）：当你想要关闭 VS Code 或断网时，需要将当前会话“挂起”到后台。 按下快捷键前缀：Ctrl + b 然后按：d 。此时你会回到原来的普通终端界面，但任务还在后台跑。

3. 断线重连

  1. 查看现有的会话：
  2. 恢复会话（Attach）： 现在你又回到了之前的界面，可以看到程序的输出进度。

常用 tmux 快捷键速查 (前缀默认为 Ctrl+b)：

  • Ctrl+b 然后 %：左右分屏
  • Ctrl+b 然后 "：上下分屏
  • Ctrl+b 然后 x：关闭当前窗格
