Polya 计数理论：从群作用到母函数

核心目标：解决在群作用下的本质不同构型（轨道）的计数问题。

贯穿示例：设 DDD 为正六边形的 6 个顶点 {1,2,3,4,5,6}\{1,2,3,4,5,6\}{1,2,3,4,5,6}，颜色集 CCC 大小为 nnn。

  • 构型集 XXX：所有可能的染色方案，大小 ∣X∣=n6|X| = n^6∣X∣=n6。

  • 群 GGG：正六边形的对称群 D6D_6D6​（二面体群），阶数 ∣G∣=12|G|=12∣G∣=12。

1. 理论基石：轨道-稳定化子定理

定义

  • 轨道 (Orbit) Ox={g⋅x∣g∈G}O_x = \{g \cdot x \mid g \in G\}Ox​={g⋅x∣g∈G}：一个特定染色方案 xxx 旋转/翻转能变成的所有样子的集合。

  • 稳定化子 (Stabilizer) Gx={g∈G∣g⋅x=x}G_x = \{g \in G \mid g \cdot x = x\}Gx​={g∈G∣g⋅x=x}：让染色方案 xxx 保持外观不变的变换集合。

定理

对于任意 x∈Xx \in Xx∈X，有：

∣G∣=∣Gx∣⋅∣Ox∣|G| = |G_x| \cdot |O_x| ∣G∣=∣Gx​∣⋅∣Ox​∣

证明引导 (Proof Intuition)

思路：建立轨道元素与陪集的一一对应。

  1. 若 g1⋅x=g2⋅xg_1 \cdot x = g_2 \cdot xg1​⋅x=g2​⋅x（两个变换把 xxx 变到了同一个状态），则 g2−1g1⋅x=xg_2^{-1}g_1 \cdot x = xg2−1​g1​⋅x=x。

  2. 这意味着 g2−1g1∈Gxg_2^{-1}g_1 \in G_xg2−1​g1​∈Gx​，即 g1g_1g1​ 和 g2g_2g2​ 属于 GxG_xGx​ 的同一个左陪集 (Left Coset)。

  3. 因此，轨道中的每一个元素对应 GGG 中 GxG_xGx​ 的一个左陪集。

  4. 由拉格朗日定理，陪集个数（指数）为 [G:Gx]=∣G∣/∣Gx∣[G:G_x] = |G|/|G_x|[G:Gx​]=∣G∣/∣Gx​∣。

2. 核心桥梁：Burnside 引理

目标

计算轨道总数 N=∣X/G∣N = |X/G|N=∣X/G∣。

定理

N=1∣G∣∑g∈G∣Xg∣N = \frac{1}{|G|} \sum_{g \in G} |X^g| N=∣G∣1​g∈G∑​∣Xg∣

其中 Xg={x∈X∣g⋅x=x}X^g = \{x \in X \mid g \cdot x = x\}Xg={x∈X∣g⋅x=x} 是在变换 ggg 作用下不动点（染色方案）的集合。

证明引导 (Double Counting)

思路：对集合 S={(g,x)∈G×X∣g⋅x=x}S = \{(g, x) \in G \times X \mid g \cdot x = x\}S={(g,x)∈G×X∣g⋅x=x} 进行双重计数。

  1. 先固定 xxx：∑x∈X∣Gx∣=∑x∈X∣G∣∣Ox∣\sum_{x \in X} |G_x| = \sum_{x \in X} \frac{|G|}{|O_x|}∑x∈X​∣Gx​∣=∑x∈X​∣Ox​∣∣G∣​。

    由于对于同一轨道 OOO 中的任意 yyy，都有 ∣Oy∣=∣O∣|O_y| = |O|∣Oy​∣=∣O∣，故 ∑x∈O∣G∣∣O∣=∣G∣\sum_{x \in O} \frac{|G|}{|O|} = |G|∑x∈O​∣O∣∣G∣​=∣G∣。

    对所有轨道求和，总和为 N⋅∣G∣N \cdot |G|N⋅∣G∣。

  2. 先固定 ggg：∑g∈G∣Xg∣\sum_{g \in G} |X^g|∑g∈G​∣Xg∣。

  3. 联立得：N⋅∣G∣=∑g∈G∣Xg∣N \cdot |G| = \sum_{g \in G} |X^g|N⋅∣G∣=∑g∈G​∣Xg∣。

3. 计算工具：Polya 计数定理 (PET)

关键转化：如何计算 ∣Xg∣|X^g|∣Xg∣？

对于置换 ggg，若要 g⋅x=xg \cdot x = xg⋅x=x（染色不变），则 ggg 的每一个循环 (Cycle) 内的所有顶点必须染同一种颜色。

设 ggg 分解为 c(g)c(g)c(g) 个不相交循环，颜色数为 nnn，则：

∣Xg∣=nc(g)|X^g| = n^{c(g)} ∣Xg∣=nc(g)

循环指标多项式 (Cycle Index Polynomial)

为了更精细地分类，定义 PGP_GPG​。设 jk(g)j_k(g)jk​(g) 为置换 ggg 中长度为 kkk 的循环个数：

PG(t1,…,t∣D∣)=1∣G∣∑g∈G∏k=1∣D∣tkjk(g)P_G(t_1, \dots, t_{|D|}) = \frac{1}{|G|} \sum_{g \in G} \prod_{k=1}^{|D|} t_k^{j_k(g)} PG​(t1​,…,t∣D∣​)=∣G∣1​g∈G∑​k=1∏∣D∣​tkjk​(g)​

定理形式

本质不同的染色方案总数（颜色数 nnn）：

N=PG(n,n,…,n)N = P_G(n, n, \dots, n) N=PG​(n,n,…,n)

5. 习题

题目：正六边形顶点用 nnn 种颜色染色。

a) 至少用 3 种颜色 (固定六边形)

b) 考虑旋转翻转 (求轨道数)

c) 考虑旋转翻转且至少用 3 种颜色

a) 至少用 3 种颜色 (Fixed Object Counting)

思路：不涉及群作用，仅组合计数。使用减法原理。

Ansa=总数−只用1色−恰好用2色Ans_a = \text{总数} - \text{只用1色} - \text{恰好用2色} Ansa​=总数−只用1色−恰好用2色

  1. 总数: n6n^6n6

  2. 只用 1 色: (n1)×1=n\binom{n}{1} \times 1 = n(1n​)×1=n

  3. 恰好用 2 色:

    • 选颜色: (n2)\binom{n}{2}(2n​)

    • 满射涂色: 26−22^6 - 226−2 (减去纯色)

    • 计算: n(n−1)2×62=31n2−31n\frac{n(n-1)}{2} \times 62 = 31n^2 - 31n2n(n−1)​×62=31n2−31n

结果:

Ansa=n6−n−(31n2−31n)=n6−31n2+30nAns_a = n^6 - n - (31n^2 - 31n) = n^6 - 31n^2 + 30n Ansa​=n6−n−(31n2−31n)=n6−31n2+30n

b) 考虑旋转翻转 (Orbit Counting)

思路：直接应用 Polya 定理，代入 tk=nt_k = ntk​=n。

Ansb=PD6(n,…,n)=112(n6+2n+2n2+4n3+3n2n2)Ans_b = P_{D_6}(n, \dots, n) = \frac{1}{12} \left( n^6 + 2n + 2n^2 + 4n^3 + 3n^2n^2 \right) Ansb​=PD6​​(n,…,n)=121​(n6+2n+2n2+4n3+3n2n2)

结果:

Ansb=112(n6+3n4+4n3+2n2+2n)Ans_b = \frac{1}{12} \left( n^6 + 3n^4 + 4n^3 + 2n^2 + 2n \right) Ansb​=121​(n6+3n4+4n3+2n2+2n)

c) 考虑旋转翻转且至少用 3 种颜色

思路：在轨道集合上使用减法原理。

Ansc=N(n)−(恰好1色轨道)−(恰好2色轨道)Ans_c = N(n) - (\text{恰好1色轨道}) - (\text{恰好2色轨道}) Ansc​=N(n)−(恰好1色轨道)−(恰好2色轨道)

  1. 恰好 1 色轨道: nnn 个 (每种纯色各构成 1 个轨道)。

  2. 恰好 2 色轨道:

    • 选 2 种颜色: (n2)\binom{n}{2}(2n​)

    • 2 色构成的非纯色轨道数: N(2)−2N(2) - 2N(2)−2 (即用2色算出的总轨道数，减去2个纯色轨道)。

    • 计算 N(2)N(2)N(2):

      N(2)=112(26+3⋅24+4⋅23+2⋅22+2⋅2)=112(64+48+32+8+4)=15612=13N(2) = \frac{1}{12}(2^6 + 3\cdot 2^4 + 4\cdot 2^3 + 2\cdot 2^2 + 2\cdot 2) = \frac{1}{12}(64+48+32+8+4) = \frac{156}{12} = 13 N(2)=121​(26+3⋅24+4⋅23+2⋅22+2⋅2)=121​(64+48+32+8+4)=12156​=13

    • 非纯色轨道数: 13−2=1113 - 2 = 1113−2=11。

    • 总项: 11×(n2)=11n(n−1)211 \times \binom{n}{2} = \frac{11n(n-1)}{2}11×(2n​)=211n(n−1)​。

计算:

Ansc=Ansb−n−112(n2−n)Ans_c = Ans_b - n - \frac{11}{2}(n^2 - n) Ansc​=Ansb​−n−211​(n2−n)

Ansc=112(n6+3n4+4n3+2n2+2n)−n−5.5n2+5.5nAns_c = \frac{1}{12}(n^6 + 3n^4 + 4n^3 + 2n^2 + 2n) - n - 5.5n^2 + 5.5n Ansc​=121​(n6+3n4+4n3+2n2+2n)−n−5.5n2+5.5n

通分 (×12\times 12×12):

Numerator=(n6+3n4+4n3+2n2+2n)−12n−66n2+66nNumerator = (n^6 + 3n^4 + 4n^3 + 2n^2 + 2n) - 12n - 66n^2 + 66n Numerator=(n6+3n4+4n3+2n2+2n)−12n−66n2+66n

Numerator=n6+3n4+4n3−64n2+56nNumerator = n^6 + 3n^4 + 4n^3 - 64n^2 + 56n Numerator=n6+3n4+4n3−64n2+56n

最终结果:

Ansc=112(n6+3n4+4n3−64n2+56n)Ans_c = \frac{1}{12} \left( n^6 + 3n^4 + 4n^3 - 64n^2 + 56n \right) Ansc​=121​(n6+3n4+4n3−64n2+56n)
