Polya 计数理论

2026 Apr 25
笔记
5 min read

核心目标：解决在群作用下的本质不同构型（轨道）的计数问题。

贯穿示例：设 $D$ 为正六边形的 6 个顶点 ${1,2,3,4,5,6}$，颜色集 $C$ 大小为 $n$。

  • 构型集 $X$：所有可能的染色方案，大小 $|X| = n^6$。
  • 群 $G$：正六边形的对称群 $D_6$（二面体群），阶数 $|G|=12$。

如果我们认为，经过群 $G$ 变换得到的不同构型 $x\in X$ 应该被视作相同的构型，那么目标就是要计算有多少个本质不同（经变换不会重合）的构型。

即计算轨道总数 $N = |X/G|$。

1. 理论基石：轨道-稳定化子定理

定义

  • 轨道 (Orbit) $O_x = {g \cdot x \mid g \in G}$：一个特定染色方案 $x$ 旋转/翻转能变成的所有样子的集合。

  • 稳定化子 (Stabilizer) $G_x = {g \in G \mid g \cdot x = x}$：让染色方案 $x$ 保持外观不变的变换集合。

定理

对于任意 $x \in X$，有：

$$|G| = |G_x| \cdot |O_x|$$

思路：建立轨道元素与陪集的一一对应。

  1. 若 $g_1 \cdot x = g_2 \cdot x$（两个变换把 $x$ 变到了同一个状态），则 $g_2^{-1}g_1 \cdot x = x$。

  2. 这意味着 $g_2^{-1}g_1 \in G_x$，即 $g_1$ 和 $g_2$ 属于 $G_x$ 的同一个左陪集 (Left Coset)。

  3. 因此，轨道中的每一个元素对应 $G$ 中 $G_x$ 的一个左陪集。

  4. 由拉格朗日定理，陪集个数（指数）为 $[G:G_x] = |G|/|G_x|$。

2. 核心桥梁：Burnside 引理

目标

计算轨道总数 $N = |X/G|$。

定理

$$N = \frac{1}{|G|} \sum_{g \in G} |X^g|$$

其中 $X^g = {x \in X \mid g \cdot x = x}$ 是在变换 $g$ 作用下不动点（染色方案）的集合。

思路：对集合 $S = {(g, x) \in G \times X \mid g \cdot x = x}$ 进行双重计数。

  1. 先固定 $x$：$\sum_{x \in X} |G_x| = \sum_{x \in X} \frac{|G|}{|O_x|}$。

    由于对于同一轨道 $O$ 中的任意 $y$，都有 $|O_y| = |O|$，故 $\sum_{x \in O} \frac{|G|}{|O|} = |G|$。

    对所有轨道求和，总和为 $N \cdot |G|$。

  2. 先固定 $g$：$\sum_{g \in G} |X^g|$。

  3. 联立得：$N \cdot |G| = \sum_{g \in G} |X^g|$。

3. 计算工具：Polya 计数定理 (PET)

关键转化：如何计算 $|X^g|$？

对于置换 $g$，若要 $g \cdot x = x$（染色不变），则 $g$ 的每一个循环 (Cycle) 内的所有顶点必须染同一种颜色。

设 $g$ 分解为 $c(g)$ 个不相交循环，颜色数为 $n$，则：

$$|X^g| = n^{c(g)}$$

循环指标多项式 (Cycle Index Polynomial)

为了更精细地分类，定义 $P_G$。设 $j_k(g)$ 为置换 $g$ 中长度为 $k$ 的循环个数：

$$P_G(t_1, \dots, t_{|D|}) = \frac{1}{|G|} \sum_{g \in G} \prod_{k=1}^{|D|} t_k^{j_k(g)}$$

定理形式

本质不同的染色方案总数（颜色数 $n$）：

$$N = P_G(n, n, \dots, n)$$

4. 实例演算：正六边形 ($D_6$) 的循环指标

我们需要遍历 $D_6$ 的所有 12 个元素，写出其循环结构。

对象集大小 $|D|=6$。

变换类型 几何描述 数量 循环分解示例 循环结构 (t 项)
恒等 旋转 $0^\circ$ 1 $(1)(2)(3)(4)(5)(6)$ $t_1^6$
旋转 $\pm 60^\circ$ 2 $(123456)$ $t_6^1$
旋转 $\pm 120^\circ$ 2 $(135)(246)$ $t_3^2$
旋转 $180^\circ$ 1 $(14)(25)(36)$ $t_2^3$
反射 对角线 (过顶点) 3 $(1)(4)(26)(35)$ $t_1^2t_2^2$
反射 对边中点 3 $(12)(36)(45)$ $t_2^3$

汇总得到 $P_{D_6}$：

$$P_{D_6} = \frac{1}{12} \left( t_1^6 + 2t_6^1 + 2t_3^2 + 4t_2^3 + 3t_1^2t_2^2 \right)$$

(注：旋转 $180^\circ$ 的 $t_2^3$ 与对边中点反射的 $3t_2^3$ 合并为 $4t_2^3$)

5. 习题

题目：正六边形顶点用 $n$ 种颜色染色。

a) 至少用 3 种颜色 (固定六边形)

b) 考虑旋转翻转 (求轨道数)

c) 考虑旋转翻转且至少用 3 种颜色

a) 至少用 3 种颜色 (Fixed Object Counting)

思路：不涉及群作用，仅组合计数。使用减法原理。

$$Ans_a = \text{总数} - \text{只用1色} - \text{恰好用2色}$$

  1. 总数: $n^6$

  2. 只用 1 色: $\binom{n}{1} \times 1 = n$

  3. 恰好用 2 色:

    • 选颜色: $\binom{n}{2}$

    • 满射涂色: $2^6 - 2$ (减去纯色)

    • 计算: $\frac{n(n-1)}{2} \times 62 = 31n^2 - 31n$

结果:

$$Ans_a = n^6 - n - (31n^2 - 31n) = n^6 - 31n^2 + 30n$$

b) 考虑旋转翻转 (Orbit Counting)

思路：直接应用 Polya 定理，代入 $t_k = n$。

$$Ans_b = P_{D_6}(n, \dots, n) = \frac{1}{12} \left( n^6 + 2n + 2n^2 + 4n^3 + 3n^2n^2 \right)$$

结果:

$$Ans_b = \frac{1}{12} \left( n^6 + 3n^4 + 4n^3 + 2n^2 + 2n \right)$$

c) 考虑旋转翻转且至少用 3 种颜色

思路：在轨道集合上使用减法原理。

$$Ans_c = N(n) - (\text{恰好1色轨道}) - (\text{恰好2色轨道})$$

  1. 恰好 1 色轨道: $n$ 个 (每种纯色各构成 1 个轨道)。

  2. 恰好 2 色轨道:

    • 选 2 种颜色: $\binom{n}{2}$

    • 2 色构成的非纯色轨道数: $N(2) - 2$ (即用2色算出的总轨道数，减去2个纯色轨道)。

    • 计算 $N(2)$:

      $$N(2) = \frac{1}{12}(2^6 + 3\cdot 2^4 + 4\cdot 2^3 + 2\cdot 2^2 + 2\cdot 2) = \frac{1}{12}(64+48+32+8+4) = \frac{156}{12} = 13$$

    • 非纯色轨道数: $13 - 2 = 11$。

    • 总项: $11 \times \binom{n}{2} = \frac{11n(n-1)}{2}$。

计算:

$$Ans_c = Ans_b - n - \frac{11}{2}(n^2 - n)$$

$$Ans_c = \frac{1}{12}(n^6 + 3n^4 + 4n^3 + 2n^2 + 2n) - n - 5.5n^2 + 5.5n$$

通分 ($\times 12$):

$$Numerator = (n^6 + 3n^4 + 4n^3 + 2n^2 + 2n) - 12n - 66n^2 + 66n$$

$$Numerator = n^6 + 3n^4 + 4n^3 - 64n^2 + 56n$$

最终结果:

$$Ans_c = \frac{1}{12} \left( n^6 + 3n^4 + 4n^3 - 64n^2 + 56n \right)$$
