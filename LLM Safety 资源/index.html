1. дёүеӨ§AIйЎ¶дјҡ & NLPйЎ¶дјҡ

йЎ¶дјҡзҡ„ж–Үз« йҖҡеёёжҜ”иҫғдёҘи°ЁпјҢејҖжәҗеҸҜеӨҚзҺ°гҖӮе№¶дё”йҖҡеёёеӯҰжңҜз•Ңзҡ„ж–Үз« е®һйӘҢжқЎд»¶жҺҘиҝ‘пјҢе®№жҳ“еҸӮиҖғгҖӮдҪҶиҙЁйҮҸе№¶йқһйғҪеҫҲеҘҪгҖӮеӣ жӯӨиҰҒз•ҷж„ҸеҲ’еҲҶзҡ„oral/spotlight/posterзә§еҲ«пјҢд»ҘеҸҠе®ЎзЁҝдәәж„Ҹи§Ғе’ҢrebuttalиҝҮзЁӢпјҢд»ҘжӯӨеҸӮиҖғж–Үз« иҙЁйҮҸгҖӮ

2. ж ёеҝғе®һйӘҢе®Ө

е·Ҙдёҡз•Ңзҡ„з ”з©¶йҖҡеёёжҜ”иҫғйўҶе…ҲпјҢеҫҲеӨҡж–№жі•е’Ңз ”з©¶ж–№еҗ‘еҖјеҫ—еҸӮиҖғж„Ҹд№үгҖӮдҪҶйҖҡеёёй—ӯжәҗгҖҒйңҖиҰҒеҫҲй«ҳи®Ўз®—иө„жәҗпјҢйҡҫд»ҘзӣҙжҺҘдёҠжүӢгҖӮ

  • Anthropic | Alignment / Interpretability (еҝ…иҜ») д»–д»¬зҡ„ж–Үз« иҙЁйҮҸжһҒй«ҳпјҢеңЁ Alignment ж–№йқўеӨ„дәҺйўҶеҜјең°дҪҚгҖӮ
  • OpenAI Safety Team еңЁйЎөйқўдёӯеҜ»жүҫ Safety ж ҮзӯҫпјҢе…іжіЁ Preparedness Framework е’Ң Superalignment (е°Ҫз®ЎеӣўйҳҹйҮҚз»„пјҢдҪҶиҝҮеҫҖж–ҮжЎЈжһҒе…·д»·еҖј) е’Ңе…ідәҺ GPT-4 System Card зҡ„жҠҖжңҜжҠҘе‘ҠпјҢйҮҢйқўиҜҰз»ҶжҸҸиҝ°дәҶзәўйҳҹжөӢиҜ•зҡ„иҝҮзЁӢгҖӮ
  • Google DeepMind зӯӣйҖү Responsibility жҲ– SafetyпјҢе…іжіЁ Safety & Ethics йғЁй—Ёзҡ„з ”з©¶пјҢзү№еҲ«жҳҜе…ідәҺ Agent Misalignment е’Ң Reward Hacking зҡ„з ”з©¶гҖӮ
  • BAIR (Berkeley AI Research) з”ұ Dan Hendrycks йўҶеҜјпјҢйқһеёёзәҜзІ№зҡ„ Safety з ”з©¶жңәжһ„гҖӮд»–д»¬еҸ‘еёғзҡ„ вҖңStatement on AI RiskвҖқ е’Ңеҗ„зұ» Safety Benchmark жҳҜеҝ…зңӢзҡ„гҖӮе…іжіЁ Center for AI Safety (CAIS)гҖӮ

1.3. зӨҫеҢәдёҺйў„еҚ°жң¬е№іеҸ°

  • йЎ¶дјҡзҡ„ Workshop еҫҖеҫҖжҜ”дё»дјҡжӣҙеүҚжІҝгҖӮеҜ»жүҫеёҰжңү вҖңGenerative AI SafetyвҖқ, вҖңTrustworthy AIвҖқ, вҖңMachine Learning SafetyвҖқ ж Үйўҳзҡ„ WorkshopгҖӮ
  • Hugging Face Papers
  • еңЁ GitHub жҗңзҙў awesome-llm-safety, awesome-ai-alignment, awesome-llm-agentsгҖӮиҝҷдәӣд»“еә“йҖҡеёёдјҡжҢүж—¶й—ҙзәҝж•ҙзҗҶеҝ…иҜ»и®әж–ҮгҖӮ
  • Alignment Forum е…ЁзҗғжңҖзЎ¬ж ёзҡ„ AI е®үе…Ёи®Ёи®әеҢәгҖӮиҝҷйҮҢдёҚеҸӘжҳҜеҸ‘и®әж–ҮпјҢжӣҙеӨҡжҳҜеҜ№жңӘжқҘйЈҺйҷ©гҖҒAgent ж¬әйӘ—гҖҒзҗҶи®әеҜ№йҪҗзҡ„ж·ұеәҰй•ҝж–Үи®Ёи®әгҖӮйҖӮеҗҲеҪ“дҪ еҜ№еҹәжң¬жҰӮеҝөзҶҹжӮүеҗҺпјҢеҺ»еҜ»жүҫ deeply insightful зҡ„и§ӮзӮ№гҖӮ
  • IEEE SaTML (Conference on Secure and Trustworthy Machine Learning): дё“жіЁдәҺе®үе…Ёе’ҢеҸҜдҝЎиө– MLгҖӮ
  • FAccT (Fairness, Accountability, and Transparency): еҒҸеҗ‘зӨҫдјҡеҪұе“ҚгҖҒе…¬е№іжҖ§е’ҢдјҰзҗҶгҖӮ
  • Xпјҡе…іжіЁжҙ»и·ғзҡ„ Safety з ”з©¶е‘ҳпјҲеҰӮ Dan Hendrycks, Paul Christiano, Neel Nanda зӯүпјүгҖӮеҫҲеӨҡи®әж–Үзҡ„дёҖдҪңдјҡеңЁиҝҷйҮҢеҸ‘ Thread и§ЈйҮҠж ёеҝғжҖқжғіпјҢжҜ”иҜ» Abstract жӣҙеҘҪжҮӮгҖӮ

2. еӯҰд№ иө„жәҗ

  • CS 324 (Stanford): Large Language Models иҜҫзЁӢпјҢе…¶дёӯжңүе…ідәҺ Safety е’Ң Harms зҡ„з« иҠӮгҖӮ
  • ML Safety Course (Dan Hendrycks): зі»з»ҹжҖ§зҡ„ ML е®үе…ЁиҜҫзЁӢпјҢж¶өзӣ–дәҶ Robustness, Monitoring, Alignment, Systemic SafetyгҖӮ

3. е…ій”®иҜҚ

  • жҗңзҙўе…ій”®иҜҚпјҡ Large Language Model Safety, Jailbreak, Hallucination, Red Teaming, Bias, Toxicity, Agent Alignment, Adversarial Attacks, Robustness, Interpretability, RLHFгҖӮ з”ұдәҺдҪ жҸҗеҲ°дәҶ Agentic LLMпјҢиҝҷжҳҜдёҖдёӘиҫғж–°зҡ„еӯҗйўҶеҹҹпјҢдё»иҰҒе…іжіЁ LLM еңЁдҪҝз”Ёе·Ҙе…·гҖҒи§„еҲ’е’Ңжү§иЎҢд»»еҠЎж—¶зҡ„йЈҺйҷ©гҖӮ
  • е…ій”®иҜҚ: Tool-use safety, Indirect Prompt Injection, Agentic Jailbreak, Power-seeking behavior, Reward HackingгҖӮ
  • е…іжіЁеҹәеҮҶжөӢиҜ• (Benchmarks):
    • еҜ»жүҫеҰӮ AgentBench, SafetyBench, Do-Not-Answer зӯүй’ҲеҜ№ Agent иЎҢдёәзҡ„иҜ„жөӢйӣҶи®әж–ҮгҖӮ
    • е…іжіЁ R-U-S (Risk-Use-Safety) зӣёе…ізҡ„жңҖж–°жЎҶжһ¶гҖӮ

4. е»әи®®зҡ„иө·жӯҘи·Ҝеҫ„

  1. е…ҲзңӢз»јиҝ° (Survey): жүҫ 1-2 зҜҮ 2024/2025 е№ҙзҡ„ Survey on Large Language Model Safety жҲ– Survey on Agentic AIгҖӮиҝҷиғҪеё®дҪ е»әз«Ӣеқҗж Үзі»гҖӮ
  2. е…іжіЁзәўйҳҹжөӢиҜ• (Red Teaming): йҳ…иҜ»е…ідәҺ LLM Jailbreaking (и¶ҠзӢұ) зҡ„ж–Үз« пјҢиҝҷжҳҜзҗҶи§Ј Agent жјҸжҙһзҡ„жңҖзӣҙи§Ӯж–№ејҸгҖӮ
  3. еҠЁжүӢе®һйӘҢ: е°қиҜ•еӨҚзҺ°дёҖдәӣз®ҖеҚ•зҡ„ Prompt Injection ж”»еҮ»жҲ–йҳІеҫЎжңәеҲ¶пјҢAgentic Safety еҰӮжһңдёҚи·‘д»Јз ҒеҫҲйҡҫзҗҶи§Је…¶дёӯзҡ„еӨҚжқӮдәӨдә’гҖӮ

рҹ’Ў з§‘з ”ж–°жүӢе®һж“Қе»әи®®пјҡеҰӮдҪ•ејҖе§Ӣз¬¬дёҖе‘Ёпјҹ

дёҚиҰҒиҜ•еӣҫжҠҠдёҠйқўжүҖжңүй“ҫжҺҘйғҪзңӢе®ҢгҖӮиҜ·жҢүд»ҘдёӢжӯҘйӘӨж“ҚдҪңпјҡ

  1. Day 1: жү“ејҖ ACL AnthologyпјҢжҗңзҙў Survey Large Language Model SafetyпјҢжүҫдёҖзҜҮ 2024 е№ҙд»ҘеҗҺзҡ„з»јиҝ°пјҲSurveyпјүпјҢеҸӘиҜ»иҝҷдёҖзҜҮпјҢжҗһжё…жҘҡйўҶеҹҹзҡ„еҲҶзұ»ж ‘пјҲTaxonomyпјүгҖӮ
  2. Day 2: еҺ» Anthropic зҡ„еҚҡе®ўпјҢиҜ»дёӨзҜҮе…ідәҺ Constitutional AI зҡ„з§‘жҷ®ж–ҮпјҢе»әз«Ӣзӣҙи§ӮзҗҶи§ЈгҖӮ
  3. Day 3: жү“ејҖ Hugging Face PapersпјҢеҸӘзңӢеҪ“еӨ©зҡ„зғӯй—ЁпјҢеҹ№е…»вҖңдҪ“ж„ҹвҖқгҖӮ
  4. е·Ҙе…·: е®үиЈ…жҸ’д»¶ Connected Papers жҲ– ElicitгҖӮеҪ“дҪ жүҫеҲ°дёҖзҜҮеҘҪж–Үз« ж—¶пјҢз”Ёиҝҷдәӣе·Ҙе…·иҮӘеҠЁеҜ»жүҫвҖңзӣёдјјж–Үз« вҖқпјҢжҜ”дҪ иҮӘе·ұжҗңиҰҒеҝ«10еҖҚгҖӮ
    еёҢжңӣиҝҷд»ҪвҖңең°еӣҫвҖқиғҪеё®дҪ е°‘иө°ејҜи·ҜпјҒеҰӮжһңеңЁйҳ…иҜ»е…·дҪ“и®әж–Үж—¶йҒҮеҲ°дёҚжҮӮзҡ„жҰӮеҝөпјҢйҡҸж—¶жқҘй—®жҲ‘гҖӮ
