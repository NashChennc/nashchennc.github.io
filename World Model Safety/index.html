StreamingT2V

StreamingSVD 是一种先进的自回归技术，用于文本转视频和图像转视频的生成，能够生成具有丰富运动动态的高质量长视频，将 SVD 转化为长视频生成器。

底层自回归方法的有效性并不局限于所使用的特定基础模型，这表明基础模型的改进可以生成更高质量的视频。StreamingSVD 是 StreamingT2V 系列的一部分。

在默认配置下，我们的代码生成 200 帧时需要 60 GB 的显存。内存优化版本显著降低显存需求至 24 GB，但运行速度比默认配置慢约 50%。为了进一步减少内存占用，可以考虑降低帧数或启用随机混合。我们的代码在 Linux 上测试，使用 Python 3.9 和 CUDA >= 11.8。

  • Website https://video-to-action.github.io

  • StreamingSVD https://github.com/Picsart-AI-Research/StreamingT2V

  • StreamingModelscope https://github.com/Picsart-AI-Research/StreamingT2V/tree/StreamingModelscope

ModelScope 服务平台，可以使用其中的 T2V api

https://github.com/modelscope/modelscope

AnimateDiff

此代码库是 AnimateDiff [ICLR2024 Spotlight] 的官方实现。它是一个即插即用的模块，可以将大多数社区文本转图像模型转换为动画生成器，无需额外训练。

  • Website https://animatediff.github.io

  • Github Repo https://github.com/guoyww/animatediff/

  • Tutorial (With Diffusers) https://huggingface.co/docs/diffusers/api/pipelines/animatediff

  • Tutorial https://cgmi.com/archives/481

LaVie

LaVie 是一个文本转视频 (T2V) 生成框架，也是视频生成系统 Vchitect 的重要组成部分。您还可以查看我们精心调优的图像转视频 (I2V) 模型 SEINE 。

  • Website https://vchitect.github.io/LaVie-project/

  • Github Repo https://github.com/Vchitect/LaVie
