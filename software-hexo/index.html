HexoCommands

  • 安装 Git 和 Nodejs(stable)

  • cmd 运行 npm install hexo-cli -g 时间略长

安装部署工具：npm install hexo-deployer-git --save

  • 查看版本 hexo -version

  • 在博客目录 git bash

  • 如果未创建：

  • 运行：本地查看 hexo s --debug
  • 部署 github

复制用户同名仓库 git https ，粘贴到 博客目录下 _config.yml，拉到最后，填写 deploy 模块：repository

  • 部署：hexo g -d
