要爱具体的人

爱抽象的人只是在爱自己。

很难说这些抽象的内容对现实毫无影响。这句话放在这里是用于警示自己，下面的内容 完全不能 也 不应该 作为任何现实情形下的判定标准。现实情况应该另当别论。

建立这个页面主要是为了剖析自己。因此为了避免削足适履，所有内容应当视作过去式。其中的内容随时间有变动，有可能和由阅读、经历、情感塑造的判断模式相关。注意时效性。通常来说，根据判定的稳定性，会有一定分层。

Part 1 当前厨力

这是本文唯一具有现实意义的部分。其现实意义仅仅在于你送我这些角色的谷我会很开心，并不代表会对现实中的这些特质有好感。

在此给出一些有建设意义的问题：

  • 印象深刻的情节？例如语言、行为等直接逻辑，考虑外观、声线是否间接反映特质。特别留意是否对相关行为对象有代入关系。
  • 注意厨力的过程。喜欢上角色时对角色的印象？增加好感度的关键节点？随着印象变化，对角色的好感的变化模式？
  • 注意自身判断模式的时效性。喜欢上角色时自己的精神状态？社交情况？审美、哲学认识情况？
  • 注意 “特质” 的交互性。对作品世界观的印象？对情节的印象？对角色之间互动关系的印象？

下面说明了一些表现形式上的不同点。如果将表现形式过于不同的角色放在一起比较，结论难免抽象化。因此必须首先给表现形式分一些大类。实际上对于不同题材甚至世界观有细微差别的作品，判断模式也会有所不同。

可以得知，实际上判断标准极难统一，因此对于厨力的归纳必须抱着审慎的态度。实际上抽象程度关系到读和写两部分，简单来说就是我可以声明“本文比起本人要更偏抽象”而写得偏抽象，也可以遇到抽象的部分就此收笔。但这篇文章很大程度上的意义在于自我剖析，因此

1. 动画

动画厨力角色的一些特点

  • 作品的剧情和表现质量有一定门槛。角色以此引起注意，这是重要的基础。出于这个原因，角色厨力很多时候只是动画内比较，多部番的横向比较现象不明显。
  • 对角色厨力的形成来源于对于观众来说的注意力节点，观众因此形成对角色行为模式、特质的深刻印象。实际上和戏份多少并不是直接关系。
  • 党争片中角色之间出现一些矛盾，这时角色的行为模式会造成区分度。有时对一部作品没有深刻印象，但角色之间区分度较大，也会引起注意。同时，党争片中的站队具有一定社交意义，或者说这属于外部因素带来的注意力。当然，如果最终没有形成足够的区分度，会有整体的印象分降低。

下面是一些厨力

  • 某些作品由于人数过多竞争太激烈惨遭淘汰，实际上如果按抽象的统一标准会入围很多人。

  • 作品厨不在表中，某些角色没有出现只是因为角色不戳 xp。

  • 没有加入传统恋爱题材作品中的角色。ちょっとはずかしい

  • 真的没有迫伞

2. 抽卡手游

抽卡手游的注意力情况就很不同。由于在持续运营的游戏中角色数量不断增长，玩家对角色需要在资源、精力上进行分配，这造成了对不同角色之间的注意力不公平。同时一定程度上造成了判断标准的浅层化。因此，抽卡手游的角色 xp 横向对比不具有一定深度的意义。

这里点名表扬崩坏 3，通过卡池中推出同角色的不同形态，在剧情上保持了相对较少的人数，甚至采取同特质角色、出演活动剧情的形式建立了类似多 if 线的效果。这样做的好处是可以在一个长的时间线甚至 if 线上描述角色的成长、变化，维持了角色塑造的厚度。目前粥也少量参考了这种模式。

然而某二字开放世界游戏没有妥善处理上述两个问题，很多时候卡池角色的剧情厚度甚至不如世界任务中的 npc。可以说这是卡池模式的一个重要问题。另外该二字游戏在角色间互动的塑造也有一定缺失，这可能和产能有关，因为如果没有选定主角团或阶段性的主角团，对每一组团体创作一定深度的剧情所需产能是巨大的。

我们不对比手游和动画的具体的塑造逻辑差异，实际上这有很多难以统一的标准和难以理清的外部因素（或者另开一坑）。各种艺术形式的表现结构本身不同、而抽卡手游的创作逻辑和接受逻辑和动画有很多根本性的不同。将不同艺术形式的艺术组成部分单拎出来比较是不负责任的。其关键在于你投入到作品中时以及日常中自己的接受形式以及对各部分的要求。

喜欢的声线

这一部分实在很难分类。主要就分为

  • 乙女。特点是：声线放松，有少量气音，听起来不完全圆润
  • 无口吐槽役（幼）。

上面两种一般来说都属于比较清晰放松的类型。似乎和平胸在声学构造上有高度关联

确实不太喜欢那种 “听起来就很大” 的声线

再次声明在几何上对平胸没有特殊爱好，可能真的是和声线有所关联

  • 无口重女。

加上这一种，上面三种基本上体现了 青涩/冷静/社恐 等要素。

另外有两大类

  • 骂人的彩喵/佐仓
  • 懒
