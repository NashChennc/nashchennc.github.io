Design

2026 Apr 4 尝试接触了一些前端设计。正好想要试用一下 Impeccable ，于是用 MDUI 2 重新设计了整个博客，添加了不少功能，变成了现在的样子。以下是我很喜欢的一些博客设计，重构的时候进行了一些参考。

  • Maxime Heckel’s Blog
  • Josh W Comeau’s Blog
  • Maggie Appleton’s Blog
  • Anthony Fu’s Blog
  • Anthropic’s Blog

我创建了这个主题的 Repo NashChennc/mdui2-blog ，不过可移植性可能不太有保证。如果你喜欢这个博客主题，也可以尝试 fork 来使用。

写博客的习惯主要还是来源于信竞时代的题解和笔记。现在想来大概是因为想要留下一些东西记录学习过程，让自己看起来好像真的在学习。最初，这个博客主题来源于 洛谷管理员 Darkflames 为洛谷博客写的 MDBlog (已成为历史) 主题。这个主题使用 MDUI 1 实现了一种 Material Design 2 风格。我非常喜欢这种扁平化的风格和 indigo-theme / pink-accent 的配色。后来在 2022 上半年，借着数据库课大作业的机会，我几乎是照抄了 MDBlog 的 html 源码写出了这样一个 UI 。

在刚写好的时候，我把这个页面当作自己的小型文件服务器的前端，用 Apache 在校园网里跑。然而总是用着用着就渐渐淡忘了。后来中途两次想要把这个博客实际利用起来。第一次（2022 Oct 6）尝试将这个服务器单独作为一个博客使用，搬运了一些以前写的洛谷博文。第二次（2023 Mar 6）把整个主题搬迁到了 Hexo，并部署到了 GitHub Pages 上，顺手清理了 自己的洛谷博客。实际上，后来的很长一段时间都没有启用过这个博客。因为后来的一年里，我开始尝试开始在完整的 $\LaTeX$ 文件内写笔记（但还是发现不如 Markdown+KaTeX 方便）。最后在 2025 Oct，在终于找到了 obsidian+hexo+sync 的可行方案之后，我终于再次重新启用这个博客，直到现在。

还记得那时都是古法 Copy Paste 写 EJS；我在折腾博客途中造了不少轮子，试了不少像 MathJax 这样的渲染插件，甚至因为插件兼容性等问题手搓过 Markdown 编译器。但后来 Vibe Coding 成熟之后这些都成为过去了。而随着 洛谷专栏 代替 洛谷博客 的生态位，MDBlog 也成为了历史。（泪目）

这张图其实是 2025 Nov 25 在 Gemini 3 Pro 发布之后，我用来尝试 Vibe Coding 做出来的。但是核心的设计语言延续了原来 MDBlog 的风格。这个配色、以及这个扁平化的设计，几乎跟 MDUI 1 文档 和许多谷歌文档 的风格一模一样。

如果你想要原本 MDUI v1 的主题，我也放在了 GitHub 上 NashChennc/hexo-mdblog 。不过也是同样不保证可移植性。或许在我有空的时候会好好维护一下。

无聊死了。来找我聊会天。
