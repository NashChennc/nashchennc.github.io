乐评：待定
Last Update: 2024/08/02 16:36
essay

一个业余小破团的弦木铜打都会一点点的乐团人决定写一点点乐评作为配器和和声的赏析练习。

古典的赏析我是水平有限啦，可以直接移步古典专辑口味展示。

想来想去还是先从原神的管弦 BGM 开始写。这份乐评最主要也不是想给乐曲一些客观全面的分析，只是想要记录一下自己听音乐时的感受。

留坑

终天的闭幕曲

璃月港主题：明月 / 千帆 / 烟火

望舒客栈：昼 / 夜

鲜衣游侠

同漩涡共舞

不冻的激浪

无虑无猜的岁月

绀田村：昼 / 夜

层岩巨渊 双轨 BGM *3：矿区 / 深层 / 岩底之钉

维摩庄：昼 / 夜

就先挖这么多坑了，HOYO-MIX 太棒了实在是没办法全都讲一遍。这里面选的是我比较喜欢的，能讲的东西不一定很多。当然发现比较有意思的有时间我会拿出来讲的。

还有一些大坑，留着，总会补的
