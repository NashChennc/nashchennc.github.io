定义 函数 f[x⃗]:∈Fn↦Ff[\vec x]:\in\mathbb F^n\mapsto\mathbb Ff[x]:∈Fn↦F 对其自变量 xix_ixi​ 的 偏微分 为

∂f∂xi∈Fn↦F:∂f∂xi[x⃗]=lim⁡h→0f[x⃗+h⋅e⃗i]h\frac{\partial f}{\partial x_i}\in\mathbb F^n\mapsto\mathbb F:\frac{\partial f}{\partial x_i}[\vec x]=\lim_{h\to 0}\frac {f[\vec x+h\cdot\vec e_i]}{h} ∂xi​∂f​∈Fn↦F:∂xi​∂f​[x]=h→0lim​hf[x+h⋅ei​]​

这样，会形成一组偏微分组

(∂f∂x1,∂f∂x2,⋯ ,∂f∂xn)[x⃗](\frac{\partial f}{\partial x_1},\frac{\partial f}{\partial x_2},\cdots,\frac{\partial f}{\partial x_n})[\vec x] (∂x1​∂f​,∂x2​∂f​,⋯,∂xn​∂f​)[x]

事实上，这个向量值函数就是 梯度 。记作 grad f\textbf{grad } fgrad f 。

此处，我们指出，定义偏微分时，我们必须认为自变量 x⃗\vec xx 的各分量不相关且完备。并且我们发现这样一组偏微分仅仅是 f:Fn↦Ff:\mathbb F^n\mapsto\mathbb Ff:Fn↦F 的定义域 Fn\mathbb F^nFn 中的一组基的基本情况 (e⃗1,e⃗2,⋯ ,e⃗n)(\vec e_1,\vec e_2,\cdots,\vec e_n)(e1​,e2​,⋯,en​) 。

自然地，我们推广得到

∂f∂v⃗i∈Fn↦F:∂f∂v⃗i[x⃗i]=lim⁡h→0f[x⃗+h⋅v⃗i]h=v⃗i⋅grad f[x⃗]\frac{\partial f}{\partial \vec v_i}\in\mathbb F^n\mapsto\mathbb F:\frac{\partial f}{\partial \vec v_i}[\vec x_i]=\lim_{h\to 0}\frac {f[\vec x+h\cdot\vec v_i]}{h}=\vec v_i\cdot\textbf{grad } f[\vec x] ∂vi​∂f​∈Fn↦F:∂vi​∂f​[xi​]=h→0lim​hf[x+h⋅vi​]​=vi​⋅grad f[x]

设 V=(v⃗1T,v⃗2T,⋯ ,v⃗nT)V=(\vec v_1^T,\vec v_2^T,\cdots,\vec v_n^T)V=(v1T​,v2T​,⋯,vnT​) ，我们有

6. 极值问题

6.1. 二元极值

对 f[x,y]f[x,y]f[x,y] 定义驻点

∂f∂x[x0,y0]=∂f∂y[x0,y0]=0\frac{\partial f}{\partial x}[x_0,y_0]=\frac{\partial f}{\partial y}[x_0,y_0]=0 ∂x∂f​[x0​,y0​]=∂y∂f​[x0​,y0​]=0

容易看出，极值点一定是驻点。再考虑其二阶偏导数，令

A=∂2f∂x2[x0,y0],B=∂2f∂x∂y[x0,y0],C=∂2f∂y2[x0,y0],A=\frac{\partial^2 f}{\partial x^2}[x_0,y_0],\quad B=\frac{\partial^2 f}{\partial x\partial y}[x_0,y_0],\quad C=\frac{\partial^2 f}{\partial y^2}[x_0,y_0],\quad A=∂x2∂2f​[x0​,y0​],B=∂x∂y∂2f​[x0​,y0​],C=∂y2∂2f​[x0​,y0​],

δ=f[x0+h,y0+k]−f[x0,y0]=12(Ah2+2Bhk+Ck2)+ο(ρ2)\delta =f[x_0+h,y_0+k]-f[x_0,y_0]=\frac 12(Ah^2+2Bhk+Ck^2)+\omicron(\rho^2) δ=f[x0​+h,y0​+k]−f[x0​,y0​]=21​(Ah2+2Bhk+Ck2)+ο(ρ2)

容易看出，如果 δ\deltaδ 是 正(负)定的，则 (x0,y0)(x_0,y_0)(x0​,y0​) 就是极小(大)值。根据中学代数知识，对如上二次型，我们有如下判别方法

令 Δ=B2−AC\Delta=B^2-ACΔ=B2−AC

若 Δ>0\Delta>0Δ>0

  • A>0A>0A>0 ，那么 δ>0\delta>0δ>0 ，(x0,y0)(x_0,y_0)(x0​,y0​) 是极小值
  • A<0A<0A<0 ，那么 δ<0\delta<0δ<0 ，(x0,y0)(x_0,y_0)(x0​,y0​) 是极大值

若 Δ<0\Delta<0Δ<0 ，那么 (x0,y0)(x_0,y_0)(x0​,y0​) 不是极值

若 Δ=0\Delta=0Δ=0 ，不能判断 (x0,y0)(x_0,y_0)(x0​,y0​) 是不是极值

线性代数中将会讨论到其他的二次型。而如果二阶导数恒为 000 ，则需要考虑更高阶的导数。在此不作讨论。
