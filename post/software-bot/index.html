安装

1. NapCatQQ

https://napneko.github.io/

打开设置 - NatCat - WebUI 即可

添加 websocketClient（反向 websocket）

ws://localhost:8080/onebot/v11/ws/

2. nonebot2

任选一个可用的 python 环境（建议虚拟机），安装脚手架用于搭建 nonebot 运行环境

cd 到一个文件夹处运行

按照引导完成

  • 项目名称: yuuko
  • 适配器: OneBot V11(0neBot V11 协议)，OneBot V12(0neBot V12 协议)
  • 驱动器: FastAPI(FastAPI 驱动器)，websockets
  • 插件存储位置: 在"src”文件夹中
  • 立即安装依赖?(Y)
  • 创建虚拟环境?(Y)
  • 无需内置插件

这样就在指定文件夹 /yuuko 处创建了一个 bot 。除了项目名称可以改，其他都不能改，因为我插件还没做适配。

创建完成后，/yuuko/.venv 是 nonebot2 插件调用的 python 解释器环境。我们在此处安装插件依赖。安装了脚手架的环境此时可以退出了。

3. 插件

安装依赖需要直接用终端

  • /.venv/Scripts/python.exe -m pip install xxx
  • 或 Linux /.venv/bin/python -m pip install xxx

总之自己找 python 解释器目录在哪。

然后将整个插件文件夹拖到 src/plugins 文件夹即可。

启动

开两个终端，分别

  • 直接运行 Lagrange.OneBot ，将 appsettings.json 中的 SignServerUrl 改为 https://sign.lagrangecore.org/api/sign/25765 ，然后扫码登录

  • 用脚手架环境 cd 到 bot 文件夹运行 nb run 即可。
