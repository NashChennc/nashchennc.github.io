1.3. 度量空间

如果映射(距离) ρ(x,y):X×X→R\rho(x,y):X\times X\to \mathbb Rρ(x,y):X×X→R

正定性ρ(x,y)≥0,且 ρ(x,y)=0⇔x=y对称性ρ(x,y)=ρ(y,x)三角不等式ρ(x,z)≤ρ(x,y)+ρ(y,z)\begin{aligned} \text{正定性} && \rho(x,y)\geq 0,\text{且 } \rho(x,y)=0\Leftrightarrow x=y\\ \text{对称性} && \rho(x,y)=\rho(y,x)\\ \text{三角不等式} && \rho(x,z)\leq \rho(x,y)+\rho(y,z)\\ \end{aligned}正定性对称性三角不等式​​ρ(x,y)≥0,且 ρ(x,y)=0⇔x=yρ(x,y)=ρ(y,x)ρ(x,z)≤ρ(x,y)+ρ(y,z)​

则称 (X,ρ)(X,\rho)(X,ρ) 为 度量空间.

范数导出度量 ρ(x,y)=∥x−y∥\rho(x,y)=\|x-y\|ρ(x,y)=∥x−y∥
